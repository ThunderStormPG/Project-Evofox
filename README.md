# ApplePriceTracker

**Live Price Comparison for Apple Products**

ApplePriceTracker is a smart e-commerce aggregator that tracks and displays real-time pricing of all Apple products (iPhone, MacBook, iPad, AirPods, Watch, etc.) across major online stores.

### Features
- Live price updates from multiple retailers
- Side-by-side price comparison
- Price history charts and lowest price alerts
- Filters by model, storage, color, and seller
- Best deal highlights with direct purchase links

Built to help users find the cheapest Apple products instantly without visiting dozens of websites.

**Focus:** Only Apple ecosystem products.
